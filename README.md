# 📦 Sistema de Caja Chica

Este es un sistema de gestión de **Caja Chica** desarrollado en **Django**, diseñado para registrar ingresos, egresos y realizar transferencias. Permite gestionar conceptos como requerimientos, extornos y gastos, además de manejar reportes y estados financieros.

## 🚀 Características
- Registro y gestión de ingresos y egresos.
- Control de rendiciones y estados de caja.
- Transferencias mediante Yape (implementación en proceso).
- Reportes detallados por fechas.
- Interfaz dinámica con **Bootstrap y Select2**.
